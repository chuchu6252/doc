入门示例

  1. 下载 链接 中的 software/hutb/hutb_car_vr_air_mujoco.zip，解压运行 CarlaUE4.exe 启动场景，按 W、S、A、D 键进行场景浏览；

  2. 安装特定 Python 版本的功能包：pip install PythonAPI/carla/dist/hutb-*.whl（或者使用pip install hutb 安装 ），然后运行 PythonAPI/examples 中的脚本（如果运行报错，请参考 解决方法）：

    • 运行 python manual_control.py --filter walker.pedestrian.* 生成键盘控制的人，运行 manual_control.py 生成一辆键盘控制的车
    • 运行 generate_traffic.py 生成人车流
    • 切换 到 VR 模式 ：config.py --map Town10HD?GAME=VR ；切换到 无人机模式 ：config.py --map Town10HD?GAME=AIR
